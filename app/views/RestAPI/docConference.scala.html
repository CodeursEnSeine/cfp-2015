Documentation for "conference"

Returns a JSON Array of Link object.

A conference is described by an eventCode, a label, a list of Locale, a localisation and a list of links to extra resources such as the list of speakers and the list of schedules.

Sample:

curl -H "Accept: application/json" @routes.RestAPI.showConference(Conference.currentConference.eventCode).absoluteURL()

Result:

{
    "label": "BreizhCamp 2015, 10th-12th June",
    "locale": [
        "en_EN"
    ],
    "localisation": "ISTIC, Campus de Beaulieu 35000 RENNES, France",
    "links": [
        {
            "href": "http://localhost:9000/api/conferences/BreizhCamp2015/speakers",
            "rel": "http://localhost:9000/api/profile/list-of-speakers",
            "title": "See all speakers"
        },
        {
            "href": "http://localhost:9000/api/conferences/BreizhCamp2015/schedules/",
            "rel": "http://localhost:9000/api/profile/schedules",
            "title": "See the whole agenda"
        },
        {
            "href": "http://localhost:9000/api/conferences/BreizhCamp2015/proposalTypes",
            "rel": "http://localhost:9000/api/profile/proposalType",
            "title": "See the different kind of conferences"
        }
    ],
    "eventCode": "BreizhCamp2015"
}
