@()(implicit lang: Lang, flash: Flash, req:RequestHeader) @api("Devoxx API") {

Devoxx API

Welcome to Devoxx France REST API.

Please, use a JSON client and discover our very simple REST API.

Each result returns an ETag. Please, check and use If-None-Match to avoid too many calls to the server.

See all events

curl -H "Accept: application/json" @routes.RestAPI.showAllConferences().absoluteURL()

See details for an event

curl -H "Accept: application/json" @routes.RestAPI.showConference(models.Conference.devoxxFrance2014.eventCode).absoluteURL()

See all speakers for this event

curl -H "Accept: application/json" @routes.RestAPI.showSpeakers(models.Conference.devoxxFrance2014.eventCode).absoluteURL()

See details for a specific speaker

curl -H "Accept: application/json" @routes.RestAPI.showSpeaker(models.Conference.devoxxFrance2014.eventCode, "25b6eeb75c18e3465d5cddf2be297b8863006551").absoluteURL()

See details for a talk

curl -H "Accept: application/json" @routes.RestAPI.showTalk(models.Conference.devoxxFrance2014.eventCode, "QDI-693").absoluteURL()

French is also supported, use Accept-language header :

curl -H "Accept: application/json" -H "Accept-language: fr-FR" @routes.RestAPI.showSpeaker(models.Conference.devoxxFrance2014.eventCode, "25b6eeb75c18e3465d5cddf2be297b8863006551")

Work in progress

I am still working on the scheduling part of Devoxx France. Slots and Agenda should be available mid-march for testing.

}
