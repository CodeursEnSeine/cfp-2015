Devoxx API

Welcome to Devoxx France REST API.

Please, use a JSON client and discover our very simple REST API.

curl -H "Accept: application/json" @routes.RestAPI.showAllConferences()
curl -H "Accept: application/json" @routes.RestAPI.showConference(models.Conference.devoxxFrance2014.eventCode)
curl -H "Accept: application/json" -H "Accept-language: fr-FR" @routes.RestAPI.showSpeakers(models.Conference.devoxxFrance2014.eventCode)

French is also supported, use Accept-language header :

curl -H "Accept: application/json" -H "Accept-language: fr-FR" @routes.RestAPI.showSpeaker(models.Conference.devoxxFrance2014.eventCode, "25b6eeb75c18e3465d5cddf2be297b8863006551")
}
